# 🎥 프로젝트 소개

<br/>
오픈소스 SW 활용 수업때 배운 장고를 실습해보기 위해서 제작된 웹페이지 이다.
문제를 출제하고 응시해 볼 수 있으며 결과창에서 응시한 채점 결과와 현재까지 누적 응시자의 평균 통계를 비교해 볼 수 있다.
Admin 기능을 통해 사용자, 문제, 정답 DB를 관리 할 수 있다.

<br/><br/>
